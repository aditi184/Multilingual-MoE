### scripts

After `wget` the model locally, to test run the hatespeech model:

In document mode:

`python run.py --infile ~/ai2/LLM/pretrain_data/c4/part_1017.jsonl.gz --outdir ~/ai2/LLM/pretrain_data/c4/ --model_name hatespeech --model_path ~/ai2/LLM/pretrain_data/hate_speech/jigsaw_fasttext_bigrams_hatespeech_final.bin --batch 100 --head 1000`


In sentence mode:

`python run.py --infile ~/ai2/LLM/pretrain_data/c4/part_1017.jsonl.gz --outdir ~/ai2/LLM/pretrain_data/c4/ --model_name hatespeech --model_path ~/ai2/LLM/pretrain_data/hate_speech/jigsaw_fasttext_bigrams_hatespeech_final.bin --level sent --sent_threshold 0.4 --batch 100 --head 10000`
